Formatting the post

  1. Let's go into our index.php file. You can see that we have our loop and a row:
      <?php while(have_posts()) : the_post(); ?>
         <div class="row">
            <div class="col-md-3">
               IMAGE HERE
            </div>
            <div class="col-md-9">
               <h2><?php echo the_title(); ?></h2>
            </div>
         </div>
      <?php endwhile; ?>
  1. Right after the row div is where we'll check to see if there is an image using if(has_post_thumbnail()) and down where the </div> ends we'll add <?php else : ?> and <?php endif; ?>.
      <div class="row">
         <?php if(has_post_thumbnail()): ?>
         <div class="col-md-3">
            IMAGE HERE
         </div>
         <div class="col-md-9">
            <h2><?php echo the_title(); ?></h2>
         </div>
         <?php else : ?>

         <?php endif; ?>
  1. If there is a thumbnail, then it's going to show a three-column and a nine-column, but if there is no thumbnail, then we want it to be a full twelve-column. For that we will use:
      <?php else: ?>
         <div class="col-md-12">
            IMAGE HERE
         </div>
      <?php endif; ?>
  1. We'll start with posts that have the thumbnail. We will get rid of the text Image Here. We will then create div with a class of post-thumbnail and then we can use the_post_thumbnail():
      <div class="col-md-3">
         <div class="post-thumbnail">
         <?php the_post_thumbnail(); ?>
         </div>
